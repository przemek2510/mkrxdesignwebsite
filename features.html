Doggo Features

Doggo has many features available for you to use!

.userinfo [@user]  | Gets information about the tagged user

.invite  | Sends an invite to the Support Server

.setuproles  | Creates roles automatically for your server

.setupchannels  | Creates channels automatically for your server

.setuptext  | Posts informational text in corresponding channels
